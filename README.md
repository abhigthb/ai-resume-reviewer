# AI Resume & Cover Letter Reviewer

Powered by xAI Grok • Built with Next.js 15 + Vercel

## Features
- PDF + TXT upload or paste text
- Resume or Cover Letter mode
- Grok AI: clarity, impact, ATS score + line-by-line rewrites
- 100% free to run (just your xAI API key)

## Deploy on Vercel
1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import repo
3. Add Environment Variable: `XAI_API_KEY` (from console.x.ai)
4. Deploy!
